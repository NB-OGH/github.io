# Nima — GitHub Pages site (Final styled bundle)
Landing page and six case studies with polished styling.

## Files
- index.html — landing page
- case-study-1.html … case-study-6.html — detailed case studies
- styles.css — responsive styling
- assets/hero.svg — hero visual

## Deploy
1. Upload all files to your `<username>.github.io` repo root and commit to `main`.
2. If GitHub Pages is already configured, it will redeploy automatically.
3. Visit `https://<username>.github.io` to verify.
